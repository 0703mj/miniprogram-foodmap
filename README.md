# 微信小程序 · 美食地图模板

美食地图小程序是一个在小程序中借助 LBS 地图功能实现获取身边美食的小程序。

## 特性

- 基于微信小程序地图实现
- 使用[小程序·云开发](https://developers.weixin.qq.com/miniprogram/dev/wxcloud/basis/getting-started.html)，无需编写后端代码
- 全移动端交互，数据管理无需再开电脑
- 独立配置项目，方便自行部署
